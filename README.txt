Show the reading time and number of words in your post.

== Description ==

Word Counter is a simple plugin that shows the reading time and number of words in your posts.

Learn more about the plugin at [Word Counter](https://wordcounter.world/tools/wordpress-plugin).

## FEATURES

- Show the reading time.
- Show the number of words.
- Decide where in the page the information will be shown.

== Installation ==

1. Upload `wordcounter` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Word Counter Settings can be accessed through the 'Settings' menu in WordPress
